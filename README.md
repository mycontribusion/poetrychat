# From Behind a Young Man's Chest - PoetryChat

An AI-powered interactive chat experience for the poetry collection **"From Behind a Young Man's Chest"** by Ahmad Musa Muhammad.

## Overview

PoetryChat allows readers to engage deeply with Ahmad Musa's poetry through a conversational AI interface. Users can ask questions about specific poems, explore themes, and discover the inspirations behind the verses.

### Features

- 🤖 **AI-Powered Insights**: Chat with an AI that has context of the entire poetry collection.
- 📱 **PWA Support**: Install the app on your mobile device or desktop for offline access and a native-like experience.
- 🎨 **Premium UI**: Modern, responsive design optimized for both mobile and desktop reading.
- 📜 **Full Collection**: Access over a hundred poems covering diverse themes including religion, family, life, and social issues.

## Project Structure

The project is split into two main components:

- `poetrychat/`: The frontend React application.
- `poetrychat-s/`: The backend Node.js/Express server handling AI chat logic.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Backend Setup (poetrychat-s)

1. Navigate to the backend directory:
   ```bash
   cd poetrychat-s
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file and add your API keys (e.g., Gemini API key).
4. Start the server:
   ```bash
   npm start
   ```

### Frontend Setup (poetrychat)

1. Navigate to the frontend directory:
   ```bash
   cd poetrychat
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
